Documentación Técnica del Lenguaje: McpMM

URI: https://www.example.org/McpMM

Prefijo: McpMM

Clase: VersionContexto

  • valorAnterior : EString (0..1)
  • fechaModificacion : EString (0..1)
  • propiedadAsociada : Propiedad (0..1)

Clase: ServiceMCP

Clase raíz que actúa como contenedor principal de todo el modelo de agentes y servicios.

  • nombreServicio : EString (0..1)
  • agentes : Agente (0..*)
  • servicios : ServiceMCP (0..*)
  • servidores : ServerMCP (0..*)
  • workflows : Workflow (0..*)

Clase: ServerMCP

Representa un servidor o infraestructura externa que expone operaciones MCP a las que los agentes pueden conectarse.

  • nombreServer : EString (0..1)
  • operaciones : OperacionMCP (0..*)

Clase: Agente

  • nombre : EString (0..1)
  • rol : EString (0..1)
  • tareas : TareaEjecutable (0..*)
  • contextos : Contexto (0..*)

Restricciones OCL:

  • R05_LimiteCargaTrabajo: self.tareas->size() <= 10

Clase: Workflow

Define el flujo de trabajo o comportamiento que sigue un agente mediante una secuencia de tareas.

  • nombreWorkflow : EString (0..1)
  • tareas : Tarea (1..*)

Restricciones OCL:

  • R01_UnaTareaInicio: self.tareas->select(t | t.oclIsTypeOf(Inicio))->size() = 1
  • R02_UnaTareaFin: self.tareas->select(t | t.oclIsTypeOf(Final))->size() = 1

Clase: Contexto

Representa el estado o conjunto de datos compartidos durante la ejecución del workflow de un agente.

  • nombreContexto : EString (0..1)
  • propiedades : Propiedad (0..*)
  • esApuntadoPor : Accion (0..*)

Clase: Propiedad

  • nombrePropiedad : EString (0..1)
  • tipoPropiedad : TipoContexto (0..1)
  • extensionPermitida : EString (0..1)
  • pesoMaxMB : EInt (0..1)
  • esUsada : Accion (0..*)
  • versiones : VersionContexto (0..*)

Clase: Tarea (Abstracta)

Clase base abstracta para todos los pasos ejecutables y puntos de control dentro de un workflow.

  • nombreTarea : EString (0..1)
  • taskId : EInt (0..1)
  • taskDesc : EString (0..1)
  • accion : Accion (0..*)
  • precede : Tarea (0..*)
  • sigue : Tarea (0..1)
  • precedeElse : TareaAnalisis (0..*)

Restricciones OCL:

  • R10_NoBuclePropio: self.sigue <> self

Clase: TareaEjecutable (Abstracta)

Clase base para aquellas tareas.

  • ejecutadaPor : Agente (1..1)

Restricciones OCL:

  • R09_EntradaYSalida: (self.precede->notEmpty() or self.precedeElse->notEmpty()) and not self.sigue.oclIsUndefined()

Clase: Inicio

Restricciones OCL:

  • R03_InicioSinEntradas: self.precede->isEmpty() and self.precedeElse->isEmpty()

Clase: Final

Nodo final que indica la terminación lógica de un workflow.

Restricciones OCL:

  • R04_FinSinSalidas: self.sigue.oclIsUndefined()

Clase: Accion (Abstracta)

Clase abstracta que modela las interacciones (lectura/escritura) que una tarea realiza sobre un contexto de datos.

  • nombreAccion : EString (0..1)
  • apunta : Contexto (1..1)
  • usa : Propiedad (0..*)

Restricciones OCL:

  • R12_CoherenciaContexto: self.usa->forAll(p | self.apunta.propiedades->includes(p))

Clase: Escritura

Acción que modifica, actualiza o inserta nuevos datos en un contexto.

Clase: Lectura

Acción que obtiene o consume datos de un contexto para su procesamiento.

Enumeración: TipoContexto

  • ENTERO = 0
  • CADENA = 1
  • REAL = 2
  • BOOLEANO = 3
  • TEXTO_EXTENSO = 4
  • ARCHIVO = 5
  • COLECCION_ARCHIVOS = 6

Clase: OperacionMCP

Define una operación específica disponible en un ServerMCP que puede ser invocada de manera remota.

  • nombreOperacion : EString (0..1)
  • descOperacion : EString (0..1)
  • esEjecutadaOperacion : TareaServerMCP (0..1)

Clase: TareaTransformacionDatos

Tarea encargada de transformar o reformatear un conjunto de datos en base a una regla o prompt.

  • prompt : EString (0..1)

Clase: TareaLLM

Tarea que invoca a un Modelo de Lenguaje Grande (LLM) utilizando un prompt específico para razonamiento o generación de texto.

  • prompt : EString (0..1)

Clase: TareaUsuario

Tarea interactiva que detiene el flujo para requerir intervención o entrada de datos manual por parte de un usuario humano.

Clase: TareaAnalisis

Tarea de bifurcación condicional que evalúa un estado para decidir continuar por el flujo principal o por un flujo alternativo (Else).

  • sigueElse : Tarea (1..1)
  • prompt : EString (0..1)

Clase: TareaEnvioContexto

Subclase de TareaEjecutable que permite exportar el contexto actual de un flujo para que sea procesado por otro agente.

Clase: TareaRecepcionContexto

Restricciones OCL:

  • R07_RecepcionPrecedidaEnvio: self.precede->exists(t | t.oclIsTypeOf(TareaEnvioContexto))
  • R11_MismasPropiedadesEnvioRecepcion: self.precede->select(t | t.oclIsTypeOf(TareaEnvioContexto))->forAll(e | self.accion->collect(usa)->asSet() = e.accion->collect(usa)->asSet())
  • R08_DistintosAgentes: self.precede->select(t | t.oclIsTypeOf(TareaEnvioContexto))->forAll(e | self.ejecutadaPor <> e.oclAsType(TareaEjecutable).ejecutadaPor)

Clase: TareaServerMCP

Tarea automatizada que se conecta a un ServerMCP para invocar remotamente una OperacionMCP específica.

  • ejecutaOperacion : OperacionMCP (1..1)
  • prompt : EString (0..1)
